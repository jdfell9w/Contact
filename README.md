# Contact XR Presentation (v6)

A cleaner presentation version of the CONTACT WebXR scene.

## Presentation approach

This version intentionally uses fewer nodes than the denser mind-map build.
The goal is to make the scene easier to read in VR and more suitable for
demonstration, explanation, or screen capture.

## Features

- cleaner node count and spacing
- original CONTACT image faintly visible as a backdrop
- large readable floating cards
- UFO sprite
- twin-Earth globe object
- richer space background
- click/select detail panel
- desktop orbit controls
- WebXR controller ray selection
- thumbstick scene shifting in VR
